# **App Name**: DevFolio

## Core Features:

- Dynamic GitHub Profile Display: Automatically fetch and present core GitHub user information, such as avatar, name, bio, and other public profile data.
- Public Repository Listing: Showcase a user's public GitHub repositories with essential details, including name, description, and direct links.
- Configurable 'About Me' Section: Provide a dedicated and editable area for users to write and display a personalized introduction about themselves.
- Work Experience Showcase: Allow users to manually add and display their professional work history, including company, role, and duration.
- Article/Blog Linkage: Enable users to include links and brief descriptions for external articles, blog posts, or publications.
- AI-Powered 'About Me' Assistant Tool: Leverage AI to suggest engaging 'About Me' section content based on the user's fetched GitHub profile data and user input.

## Style Guidelines:

- Primary color: A sophisticated, deep violet-blue (#442966) for a modern, professional, and tech-inspired feel. HSL: 260, 60%, 40%.
- Background color: A subtly tinted light gray (#F3F1F5) with a hint of violet, providing a clean and inviting canvas for content. HSL: 260, 15%, 95%.
- Accent color: A vibrant, clear blue (#5880F7) used for interactive elements, highlights, and calls to action, drawing attention to key features. HSL: 230, 80%, 60%.
- All text will use 'Inter', a grotesque-style sans-serif font, for its modern, clean, and highly readable qualities across all screen sizes.
- Utilize a consistent set of minimal, outline-style icons to represent GitHub, email, social media links, and different profile sections, ensuring a polished look.
- Implement a responsive, multi-column layout for larger screens, transitioning to a single-column stack on smaller devices, prioritizing content readability and accessibility. Content is clearly divided into distinct, easily navigable sections.
- Incorporate subtle, tasteful animations for page transitions, hover states on interactive elements, and loading indicators to enhance the user experience without being distracting.
